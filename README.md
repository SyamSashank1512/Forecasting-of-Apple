# Apple Dataset Forecasting

This project contains a Jupyter Notebook and a Streamlit application used for forecasting Apple stock prices using LSTM neural networks.

## Files Structure

- `Final_Apple.ipynb`: Jupyter notebook for data visualization, exploration, and model building.
- `lstm.py`: A Streamlit app that runs an LSTM model to forecast stock prices.
- `requirements.txt`: Python dependencies required to run the project.

## How to run the Streamlit App

1. Install the requirements
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application
   ```bash
   streamlit run lstm.py
   ```
